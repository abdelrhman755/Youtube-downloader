# STARK - Youtube Downloader

A clean and simple YouTube video downloader website built with HTML, CSS, and JavaScript.

## Live Demo
https://stark-converter.netlify.app

## Description
STARK Youtube Downloader is a web application that allows users to enter a YouTube video link, 
validate it, preview the video thumbnail, and get redirected to a download page to download the video.

## Features
-  YouTube link validation
-  Video thumbnail preview
-  Redirect to download page
-  Supports both youtube.com and youtu.be links
-  Enter key support
-  Clean and responsive design

## How To Use
1. Open the website
2. Paste a YouTube video link in the input box
3. Click **Download** or press **Enter**
4. Wait for validation
5. Click **Open download page** to download the video
